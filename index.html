Privacy-first by design.

TapFix AI is designed to work only when you ask it to. You stay in control of where the assistant appears.

Accessibility Service: used to detect editable text fields, show the floating AI button, read selected text after your action, and insert corrected, translated or rewritten text back into the active input field.
Overlay permission: used to show the floating AI button above allowed apps.
GPT API processing: selected text may be sent to an AI API only to generate the result requested by the user.
User control: choose allowed apps and disable permissions anytime in Android Settings.
